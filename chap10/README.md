## 예외 처리

### 예외 클래스  // chap10.sec01

#### 예외 : 사용자의 잘못된 조작, 개발자의 잘못된 코딩으로 인해 발생하는 프로그램 오류

<pre>
  1. 일반 예외(컴파일 예외) : 컴파일 과정에서 발생하는 예외
  2. 실행 예외 : 실행 시 발생하는 예외
</pre>

##### 모든 예외는 java.lang.Exception 클래스를 상속받는다.



### 예외 처리    // chap10.sec02

#### 예외 처리 코드

<pre>
  <code>
    try {
        [예외 발생 가능 코드]
    }
    catch(예외 클래스 e) {
        [예외 처리]
    }
    finally {
        [항상 실행]
    }
  </code>
</pre>
