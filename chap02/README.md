### 변수와 타입

#### 변수 선언
<pre>
  [타입] [변수명];
</pre>

#### 타입의 종류

##### 1. 기본 타입

  |  | 1byte | 2byte | 4byte | 8byte |
  | ----- | ----- | ----- | ----- | ----- |
  | 정수형 | byte | short, char | int | long |
  | 실수형 |      |        | float | double |
  | 논리형 | boolean | | | |

##### 2. 참조 타입 ( = 객체)  // chap05

<pre>
  기본 타입(8개) 를 제외한 나머지 타입  
</pre>

##### 3. 차이점

<pre>
  기본 타입 → 값을 저장
  참조 타입 → 번지(주소)를 저장
</pre>
